# Experimental Work - P-TEFb Regulation Project

This directory contains all experimental components of the PhD work:

## Yeast Genetics
- Error-prone PCR mutagenesis of Cyclin T1
- Reverse two-hybrid screen for HEXIM1-binding loss
- Forward two-hybrid validation

## Molecular Cloning
- Construction of Cyclin T1 and HEXIM1 alleles
- Fusion constructs (Gal4, Dendra)

## Mammalian Cell Assays
- HIV LTR luciferase reporter assays
- Gal4-CycT1 activation
- Tat rescue assays

## Biochemistry
- Co-immunoprecipitation of CyclinT1/CDK9/HEXIM1/Tat
- Western blot quantification
- CTD kinase activity measurements

## Imaging
- Dendra-CycT1 localization
- Nuclear FLIP mobility assays
- Single-molecule tracking
