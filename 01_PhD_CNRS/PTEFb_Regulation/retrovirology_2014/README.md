# Retrovirology 2014 — CyclinT1 Mutations Affecting HEXIM1 and Tat Binding

This directory gathers the experimental work underlying the publication:
“A Cyclin T1 point mutation that abolishes P-TEFb binding to HEXIM1 and HIV-1 Tat.”

## Content

### Yeast Genetics
- Reverse two-hybrid screening (loss of HEXIM1 binding)
- Forward two-hybrid validation
- Plasmid maps and mutagenized constructs

### Mammalian Cell Assays
- Luciferase reporter assays (HIV LTR / Gal4 fusions)
- CTD kinase activity assays
- Tat rescue assays
- Co-immunoprecipitation and Western blot detection

### Imaging
- Localization of CyclinT1 mutants (Dendra fusion)
- FLIP mobility measurements
- Single-molecule tracking in living cells

### Data
- Raw quantifications
- Gel scans and microscopy source files
- Summary tables for characterized mutants
